C++ Code Submission Checklist

Home About Technical Personal Fanfiction

Header Comments: Name of the File, Whay's the file about, Name of the Author

Parameter passing: pass by value, pass by reference, or pass by const reference.

Function returning: pass by value, pass by reference, or pass by const reference.

Functions: function prototype.

Files: opening check (exit(1)), input.close() at the end.

Heap: delete contents before change the pointer.

Pointer: p = nullptr after delete p (if not pointing anywhere).

Error message: output error message and/or return true/false.

Copy control: overload the four together - constructor, destructor, copy constructor, assignment operator.

Methods: when not changing the contents, make it const.

Block comments: Struct, Loops, Functions (about, parameters, returns)
